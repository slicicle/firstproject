My HTML Cheatsheet!

Table Tags

Tag Name Description
<table> Table Wrapper element for all html tables.
<thead> Table Header Row The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<th> Header Cell The header cell for a group of table cells.
<tr> Table Row A table row container.
<td> Table Data Cell A table cell containing data.
<tfoot> Table foot The set of rows defining the footer table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
border Border Sets weight and style of a table borders.
